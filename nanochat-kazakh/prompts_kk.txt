# Evaluation prompts for Kazakh language model

# Simple continuations
Қазақстан — бұл
Бүгін ауа райы
Менің атым

# Questions
Қазақ тілінде неше дауысты дыбыс бар?
Астана қаласы қашан астана болды?
Қазақстанның ең үлкен қаласы қай қала?

# News style
Президент бүгін халыққа үндеу жасап,
Экономика министрлігінің мәліметінше,
Ғалымдар жаңа зерттеу нәтижелерін

# Literature style
Алыстағы ауылда бір кәрі шал тұратын,
Күн батып, аспан қызыл түске боялды.
Жігіт атқа мініп, далаға шықты.

# Science style
Жасанды интеллект технологиялары
Қазіргі заманғы физика ғылымында
Математикалық модельдеу арқылы

# Conversational
Сәлеметсіз бе! Мен сізге
Қайырлы күн! Бүгін біз
Рахмет! Сіздің көмегіңіз
