# Scripts

Utility scripts organized by purpose. These are one-off or semi-manual scripts, NOT part of the `slm` package.

## Structure

| Directory | Purpose |
|-----------|---------|
| `data/` | Dataset preparation, cleaning, balancing, collection |
| `tokenizer/` | Tokenizer training, tokenization, upload to HF Hub |
| `training/` | Training launch scripts, MoE upcycling, multi-GPU setup |
| `eval/` | Evaluation, benchmarks, arena, routing analysis |
| `inference/` | Inference demos, GEC inference |
| `translate/` | Kazakh-Russian translation (Gazeta dataset, MADLAD) |
| `hub/` | HuggingFace Hub publishing, migration, renaming |
