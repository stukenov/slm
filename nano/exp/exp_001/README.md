# exp_001 — XOR MLP на tinygrad

## Задача
Запустить самый простой пример обучения и инференса на tinygrad — проверить что фреймворк работает.

## Контекст
Первый эксперимент в nano/. Пользователь попросил "базово просто запустить в демо варианте супер микрообучение на tinygrad и сделать инференс".

## Архитектура
- 2-слойный MLP: Input(2) → Linear(16) + ReLU → Linear(1) + Sigmoid
- Loss: Binary Cross-Entropy
- Optimizer: SGD (lr=1.0)
- Задача: XOR (4 примера, нелинейно разделимая)

## Результаты
- 500 шагов, loss: 0.70 → 0.001
- Инференс идеальный: 0⊕0=0.001, 0⊕1=0.999, 1⊕0=0.999, 1⊕1=0.001

## Выводы
- tinygrad API похож на PyTorch, но есть отличия: `Tensor.training=True`, `get_parameters()`
- Фреймворк работает, можно переходить к более сложным задачам

## Запуск
```bash
cd nano && .venv/bin/python exp/exp_001/train.py
```
